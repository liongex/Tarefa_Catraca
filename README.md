# Tarefa_Catraca

Repositório destinado ao desenvolvimento de software embarcado para controle de catraca eletrônica.

## Sobre o Projeto

Este projeto foi elaborado para fins acadêmicos ou de estudo em sistemas embarcados. Ele utiliza a linguagem C para implementação principal e CMake para gerenciar o processo de build. Este projeto foi feito usando o PICO-SDK

## Estrutura do Projeto

- **src/**: Código fonte do projeto em C.
- **CMakeLists.txt**: Script para automação de build com CMake.
- **README.md**: Documento descritivo e orientativo do projeto.

## Tecnologias Utilizadas

- **C**: Linguagem de programação central do projeto.
- **CMake**: Ferramenta para automação do processo de compilação.
